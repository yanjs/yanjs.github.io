2. Using the Python Interpreter 使用Python解释器

2.1. Invoking the Interpreter

Linux解释器一般在 /usr/local/bin/python3.7 或 /usr/local/python 要不然找一下。

Windows别忘了在命令行设置环境变量，分号后路径为解释器所在路径 set path=%path%;C:\python37 Windows打开Python解释器方法

（Ctrl+D = ^D = c-d）

进入交互式解释器方法：键入python

退出解释器方法：在一行仅有 >>> 提示符时， c-d c-z + Enter quit() 三选一（不一定都行）。

按c-p，若不显示^P，则拥有命令行编辑功能。

解释器像Shell一样工作。

在命令行中：

python -c command [arg] ...

command ：要执行的命令

python -m module [arg] ...

module ：直接执行的模块，模块必须要有源文件，如 sys 模块就没有源文件。

python -i *.py

执行脚本后进入交互模式

命令行选项大全，也可找man python 或 python --help。

3.1. Using Python as a Calculator 把Python用作计算器

3.1.1. Numbers 数字

_ 相当于计算器上的ans，保留了最后一次计算结果。赋值语句不算计算结果。_ 属于内置魔法变量，不要尝试显式赋值给它，不然会创建一个同名本地变量将其屏蔽，失去了魔法。

Python不仅支持int和float型，还支持Decimal和Fraction型（需要import相应库），竟然还支持复数（内置功能，无需import）。使用方法： 3+5j 表示复数 3+5i ，j 是虚数单位。

3.1.2 Strings 字符串

在字符串前面加 r 可以不使用任何转义，多用于路径和正则表达式。

""" 或 ''' ：三引号。成对的三引号可以随意换行，并自动将 \n 换行符加入字符串。若不想在某行结尾自动被添加换行，可以在行尾键入 \ 。

下标索引切片示例：

 +---+---+---+---+---+---+
 0   1   2   3   4   5   6
-6  -5  -4  -3  -2  -1

（部分简单使用，如灵活使用 ' 和 " 略）
